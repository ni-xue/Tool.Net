针对于新版Ashx路由模式，的同步，异步，支持

该控制器，相对于Mvc的控制器轻，应有功能都有，可自由扩展。

方便实现，您最想实现的效果，最大的优点还是因为他轻量级。

Api 方法创建示例： 
public  GetApi( context)  => Json(new { msg = "路由版本api。" });
Api 方法创建示例：(异步实现) 
public async  GetTaskApi( context) => await JsonAsync(new { msg = "路由版本api。" });
